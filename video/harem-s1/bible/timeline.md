## 관계 변화 추적

### Tasha → Ethan

| EP | 감정 | 행동 |
|----|------|------|
| 01~02 | 귀여운 동생 (가족 애정) | Full-body hug, 이불 속 침투, 감고 자기, 먹여주기. 본인은 "가족이니까" |
| 03 | 보호자 모드 (경계심 ON) | Serena+Lily가 밀착하자 Ethan 끌어당김 → 아직 질투 아닌 보호 |
| 04 | 경계 심화 | Serena 과외에 불편함, 그러나 직접 행동은 아직 |
| 05 | 질투 자각 (전환점) | Mika 무릎 앉기 목격 → 접시 기울어짐. 밤에 란제리 고르기 = "가족 사랑"이 아님을 자각 |
| 06 | 적극 공세 | 앞치마만 착용, 이마 뽀뽀, 허리 감싸기 → 의식적 유혹 시작 |
| 07 | 확신 접근 | 비키니, 선크림 요청, 넘어짐 사고 후 Ethan 입술 응시 |
| 08 | 진심 확인 | 담요+위로. Ethan이 먼저 손 잡아줌 → 확신 |
| 09~10 | 확정 | 진지한 방문. Ethan 수용. |

### Mika → Ethan

| EP | 감정 | 행동 |
|----|------|------|
| 01~03 | 무관심 (그냥 남동생) | 비키니/수건/스포츠브라 무신경, 눈앞에서 옷 갈아입어도 아무렇지 않음 |
| 04 | 변화 시작 | Serena+Lily가 Ethan에게 밀착하는 걸 목격 → 화장실 앞 kabedon = 처음으로 "빼앗기기 싫다" |
| 05 | 의도적 도발 | 게임 벌칙 무릎 앉기 = 무의식→의식 전환. 짧은 옷 의도적 선택 |
| 06 | 경쟁 본격화 | Ethan 허벅지에 머리 눕힘, 손을 자기 배에 올려놓음 → 적극 |
| 07 | 물리적 충돌 | 물속에서 다리 감김 → "Oops" = 천진함 유지하지만 떨어지기 싫어함 |
| 08 | 상처 | Ethan+Tasha 손잡기 목격 → 문틀 쥐어짐 + 눈물 → 첫 좌절 |
| 09 | 폭발 | 방문 벌컥, 손목 잡기, 눈물, 입술 거리 직전 → 천진했던 Mika의 완전한 변화 |
| 10 | 수용 | Ethan 양팔 벌리면 함께 안김 |
