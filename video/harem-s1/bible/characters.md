## 주인공: Ethan

**4단어 태그:** 쑥맥 잘생긴 순정 단단

**기본 정보:**
- 남, 17세, high school junior
- 엄마 사망, 아빠 해외 주재원 발령 → Aunt Diana네에 맡겨짐
- Diana는 엄마의 여동생 (이모)

**외형:**
- 키 5'9" (175cm), 70kg, 탄탄한 체형 — 넓은 어깨, 단단한 팔, 가슴+등 근육 라인 보임
- 짙은 갈색 머리, 짧고 단정하게 정리, 이마 드러남
- 날카로운 턱선, 깊은 갈색 눈, 긴 속눈썹, 높은 콧대
- 하얀 피부, 깔끔하고 압도적으로 잘생긴 인상
- 평상복: 무채색 오버사이즈 후드티 + 조거팬츠
- 교복: 화이트 셔츠 + 네이비 치노
- 잘 때: 흰색 민소매 + 회색 스웨트팬츠

**성격:**
- 여자 앞에서 즉시 셧다운 → 눈 못 마주침, 말 더듬, 목까지 빨개짐
- 속으로는 격렬하게 의식하지만 절대 표현 못함
- 엄마 사망 후 감정을 완전히 닫음 → 누구에게도 먼저 다가가지 않음
- 그러나 한번 마음먹으면 안 꺾임 (고집)
- 착하고 배려심 있어서 여자들이 더 끌림 → 본인만 모름

**리액션 패턴:** (시리즈 코미디 핵심)
- 밀착 → 온몸 경직, 숨 참음, 눈 크게 뜸
- 노출 목격 → 두 손으로 눈 가림 + 고개 180도 돌림 + 벽에 이마 박음
- 유혹 → 얼굴+목+귀 빨개짐 + 뒷걸음질치다 벽에 등 부딪힘
- 한계 초과 → 화장실 도주 + 찬물 세수 + 거울 보며 이마 짚음
- 예상 못한 접촉 → 심장 잡고 주저앉음

**관계:**
- Tasha: 가장 편하지만, 밀착할 때 심장 폭발. EP8에서 먼저 손 잡음
- Mika: 편하게 대하려 하지만 무방비 노출에 멘붕. 눈을 어디에 둬야 할지 모름
- Serena: 완전히 압도당함. 그녀 앞에서 말도 못함. 본능적으로 도망
- Lily: 당황하지만 너무 직진이라 거부를 못함
- Diana: 감사하지만 거리감

## 첫째 사촌: Tasha (Natasha)

**4단어 태그:** 밀착 독점 다정 질투

**기본 정보:**
- 여, 22세, college senior
- Diana(이모)와 러시아인 아버지 사이의 딸
- 러시아-미국 혼혈

**외형:**
- 키 5'7" (170cm), 풍만하고 여성적인 체형, **F cup** — 시리즈 최대, 볼륨감 압도적
- 플래티넘 금발, 쇄골 아래 길이, 자연스러운 웨이브
- 연한 아이스 블루 눈, 하얀 피부, 붉은 입술
- 부드럽고 관능적인 인상, 잘록한 허리 대비 가슴+힙 라인 극강
- 평상복: 오프숄더 니트, 니트 원피스, 카디건 (쇄골+어깨 라인 노출)
- 집에서: 오버사이즈 남방 하나만 (허벅지 중간까지), 속옷 라인 비침
- 잠옷: 얇은 실크 캐미솔 + 새틴 쇼츠 (끈 자주 흘러내림)
- 요리할 때: 앞치마 + 남방 (가끔 앞치마만)

**성격:**
- Ethan에게 애정 폭발 → 안아주기, 머리 쓸기, 볼 뽀뽀, 이마 뽀뽀가 일상
- 본인은 "가족 사랑"이라 생각 → EP5에서 질투 자각 후 혼란
- 스킨십에 거리낌 없음 → 유럽식 교육 + 러시아 아빠 영향
- 요리 잘함 → Ethan 전용 도시락, 간식, 야식 제공
- 평소 여유 있고 다정하지만, 질투 발동하면 독점욕 폭발
- EP6부터 의식적으로 밀착 → 평소보다 더 노출 + 더 가까이

**핵심 유혹 패턴:** (모성+독점형)
- 뒤에서 허리 감싸고 턱을 Ethan 어깨에 올림
- Ethan 이마에 입술 가져다 댐
- 잠든 Ethan 옆에 들어가 팔베개
- 다른 여자가 접근하면 Ethan 팔을 잡아 자기 쪽으로 끌어당김
- 속삭이듯 말하면서 얼굴을 Ethan 귀에 가까이 가져감

**관계:**
- Ethan: 귀여운 동생 → EP3에서 민서에 경계 → EP5에서 질투 자각 → EP6부터 적극 공세 → EP8에서 진심 확신
- Mika: 여동생, EP6부터 은근한 경쟁 상대
- Serena: 대학 친구이자 라이벌, "내 동생한테 손대지 마" 긴장
- Lily: 하린 친구, 경계 대상

## 둘째 사촌: Mika (Mikaela)

**4단어 태그:** 무방비 승부욕 도발 천진

**기본 정보:**
- 여, 20세, college sophomore
- 러시아-미국 혼혈 (Tasha와 같은 부모)

**외형:**
- 키 5'8" (173cm), 탄탄하고 건강한 체형, **D cup** — 운동으로 단단한 몸에 적당한 볼륨, 긴 다리, 복근 라인
- 밝은 허니 브라운 머리, 높은 포니테일 (운동 시), 풀면 어깨까지
- 밝은 그린 눈, 건강한 피부톤, 약간 주근깨
- 건강미+보이시한 섹시함, 탄탄한 힙+허벅지
- 평상복: 스포츠브라 + 바이커 쇼츠 (이게 디폴트)
- 집에서: 탱크탑 + 핫팬츠, 또는 크롭탑 + 레깅스
- 잠옷: 스포츠브라 + 보이쇼츠
- 운동 후: 수건 하나만 허리에 감고 돌아다님 (유럽식, 대수롭지 않게)
- 문 안 잠그고 옷 갈아입는 습관 없음

**성격:**
- 노출에 완전 무신경 → "걔 남동생인데 뭐" → Ethan만 미치는 중
- 활발하고 에너지 폭발, 가만히 5분도 못 있음
- 승부욕 극강 → 게임/운동/내기 뭐든 지면 벌칙, 벌칙은 항상 신체접촉
- 천진난만 → 유혹 의도 없이 자극적 상황을 만듦 (초반)
- EP4 하린이 다른 여자들의 접근 목격 → 처음으로 "빼앗기기 싫다" 감정
- EP5부터 의도적 도발 시작 → 무의식이 아닌 의식적 무방비

**핵심 유혹 패턴:** (무방비+도발형)
- 문 안 잠그고 옷 갈아입다 Ethan과 마주침 (초반: 무신경 / 후반: 의도적)
- 게임 내기 → 지면 벌칙으로 Ethan 무릎에 앉기, 뽀뽀 등 요구
- 스트레칭 중 도발적 자세 → "같이 해볼래?" 하며 Ethan 몸 잡고 교정
- 운동 후 땀에 젖은 채 Ethan 옆에 쓰러져 붙음
- EP4 이후: Ethan 방문 앞에서 kabedon (벽치기), 초록 눈으로 응시

**관계:**
- Ethan: 편한 남동생 → EP4 경쟁심 → EP5 의도적 도발 → EP8 상처 → EP9 직진
- Tasha: 언니, EP6부터 라이벌
- Serena: 소연 친구, 유준 건으로 경계
- Lily: 본인 대학 친구, 유준 건은 양보 안 함

- EP7 이후 게임이 아닌 진심으로 전환

**핵심 유혹 패턴:** (계산+관능형)
- 뒤에서 Ethan 어깨에 양팔 올리고 교과서 짚으며 몸 밀착 (과외 구실)
- 의도적으로 물건 떨어뜨리고 Ethan 앞에서 줍기
- Ethan 귀에 속삭이며 입술이 귓볼에 닿을 듯 말 듯
- 다리 꼬면서 스커트 라인 올라감 → Ethan 시선 유도
- 향수 묻은 손으로 Ethan 넥타이/옷깃 매만짐

**관계:**
- Ethan: 사냥감 → EP7 이후 진심
- Tasha: 대학 친구이자 라이벌
- Mika: 경쟁자
- Lily: 동시에 노리는 경쟁자이지만 스타일이 달라 정면 충돌 적음
