## 1. 쇼츠 시리즈 드라마 트렌드

### 포맷 현황
- 40~50초 쇼츠가 TikTok·YouTube Shorts·Reels의 최적 구간
- 시청자의 스와이프 판단: 3초 이내. 첫 프레임이 전부
- 10편 이내 미니 시리즈: 빈지 유도에 최적. 총 러닝타임 ~7분 = 한 번에 몰아볼 수 있는 분량

### 성공하는 쇼츠 시리즈의 공통점
1. **1에피소드 = 1장면 = 1감정**: 40~50초 안에 하나의 감정 변화만 전달
2. **매 편 끊기**: 해결 없이 긴장 상태로 끊어서 다음 편 유도
3. **관계 중심 서사**: 설정 설명 없이 캐릭터 간 관계 변화로 이야기 전달
4. **비주얼 내러티브**: 대사보다 표정·조명·색감으로 감정 전달

### 안티히어로 / 도덕적 딜레마 트렌드
- "착한 주인공이 어쩔 수 없이 나쁜 일을 한다" 구조가 쇼츠에서 높은 공감 획득
- 시청자 반응: "나라도 그랬을 거야" → 공유 + 댓글 유도
- 핵심: 주인공의 선택에 도덕적 대가가 따라야 감정적 무게가 생긴다
- 성장 = 강해지는 것이 아니라 소중한 것을 포기하는 것

## 2. 고양이 캐릭터 AI 애니메이션 트렌드

### 바이럴 요소
- AI 동물 영상이 TikTok·Instagram Reels·YouTube Shorts를 지배하는 추세
- 고양이 얼굴 + 인간 몸 포맷: 감정 표현이 풍부하면서도 인간 서사에 자연스럽게 이입
- 감성+서사 조합이 단순 유머보다 구독 전환에 유리

### 캐릭터 일관성 과제
- AI 생성의 최대 난제: 에피소드 간 동일 캐릭터 유지
- 해결 전략: 동일한 캐릭터 설명 프롬프트를 매 생성마다 재사용
- 소규모 캐스트(3~5명)가 일관성 유지에 유리

### 우리 채널 적용
- 고양이 얼굴 + 인간 몸: 감정 표현 + 바이럴 잠재력 겸비
- 묘종으로 캐릭터 구분 → 시각적 개성 확보
- 10편 시리즈 = 캐릭터 수 최소화 가능 = AI 일관성 확보 용이

## 3. 10편 미니 시리즈 아크 구조

### 포맷 기본
- 에피소드 길이: 40~50초
- 시리즈 편수: 10편
- 총 러닝타임: ~7분
- 매 에피소드 마지막 5초: 해결되지 않은 긴장으로 끊기

### 에피소드 내부 구조 (40~50초)
| 구간 | 시간 | 기능 |
|------|------|------|
| Hook | 3~5초 | 즉각적 시각 자극. 스와이프 방지 |
| Development | 30~40초 | 하나의 감정 변화 또는 관계 전환. 예상을 깨는 전환 1회 |
| Unresolved End | 마지막 5초 | 해결 없이 끊는다. 긴장 상태 유지 |

### 3-Act 구조 (10편 기준)
| Act | 에피소드 | 서사 기능 |
|-----|----------|-----------|
| Act 1 | EP 01~03 | Setup: 상황 제시, 주인공의 선택, 첫 도덕적 대가 |
| Act 2 | EP 04~07 | Escalation: 관계 균열, 점점 깊어지는 수렁, 핵심 반전 |
| Act 3 | EP 08~10 | Cost & Resolution: 대가 청구, 최종 선택, 여운 |

## 4. 바이럴 전략

### 에피소드 = 바이럴 단위
- 40~50초 에피소드 자체가 YouTube Shorts / TikTok / Reels 네이티브
- 추가 바이럴 티저: 에피소드에서 10~15초 핵심 장면 추출 → 크로스 플랫폼

### 바이럴 최적화 원칙
- 첫 3초: 시각적 훅 (표정, 조명, 비주얼 임팩트)
- 해시태그: 3~5개
- 10편 시리즈 구조 = 빈지 유도. 1편이 바이럴 → 나머지 9편 자동 소비
- "이 다음에 어떻게 돼?" 댓글 유도 = 알고리즘 부스트

### 바이럴 티저 추출 기준
1. 감정 폭발 (눈물, 분노, 충격 표정)
2. 관계 전환 순간 (우정 파괴, 배신, 재회)
3. 비주얼 임팩트 (조명 변화, 극적 구도)
4. 강렬한 대사 한 줄

## 5. AI 애니메이션 제작 특성 (Kling 기준)

### 40~50초 에피소드 = 3~5샷 구성
- Kling 최적 샷 길이: 5~15초
- 에피소드 1편 = 3~5개 샷 = 1회 Kling 생성으로 완성 가능
- 이 포맷은 AI 생성 최적 구간과 자연스럽게 일치

### 캐릭터 일관성
- 레퍼런스 이미지 사용 시 안정
- 고양이 얼굴 특징(귀, 무늬, 눈 색)을 프롬프트에 매번 포함하면 일관성 확보 용이
- 소규모 캐스트(3~5명)가 일관성에 유리

### 분위기 연출 (빛·색감·분위기)
- Kling은 조명·분위기 연출에 강점: 네온 불빛, 비 오는 거리, 따뜻한 실내 조명
- 세계관을 대사가 아닌 비주얼 분위기로 전달하는 데 적합
- 카메라 앵글·조명 지시가 프롬프트에서 잘 반영됨

### 서사 규칙으로 변환
| 기술 특성 | 서사/연출 규칙 |
|-----------|--------------|
| 5~15초 샷 최적 | 씬당 10~15초. 컷 전환으로 연결 |
| 다수 캐릭터 시 왜곡 가능 | 핵심 캐릭터 1~2명 중심. 나머지는 다음 샷이나 음성으로 |
| 조명·분위기 연출 강점 | 세계관을 빛·색감·날씨로 표현. 설명 대사 대신 비주얼 |
| 물리 시뮬레이션 제한적 | 비·연기 등은 분위기 조명과 사운드로 연출 |
| 전신/상반신 최적 | 감정은 상반신 클로즈업. 공간감은 미디엄 샷 |

## 6. Kling 프롬프트 작성법 (style_guide.md 기반 리서치)

> 조사일: 2026-03-02
> 목적: bible/style_guide.md 작성을 위한 Kling 프롬프트 구조·일관성·분위기 연출 리서치

### 프롬프트 구조

최적 프롬프트 공식 (30~60단어):
```
[샷 타입] + [캐릭터/피사체] + [동작] + [환경/배경] + [카메라 움직임] + [조명/무드] + [스타일]
```

- 30~60단어가 최적. 50단어 초과 + 스타일 키워드 과적재 시 결과 흐려짐
- `++핵심 요소++` 구문으로 중요 요소 강조 가능
- 시간 순서: `"initially... then... finally"` 로 클립 내 진행 지시 가능
- `"focus on"`, `"centered on"` 으로 시선 유도
- 전경/중경/배경을 분리 기술하면 깊이감 향상

### 네거티브 프롬프트

표준 세트:
```
blur, distortion, watermark, text overlay, low quality, flickering,
morphing faces, extra limbs, bright colors, cartoon, anime, pixar,
smiling, oversaturated, plastic look, CGI artifacts, inconsistent lighting
```

핵심: "no robots" 가 아니라 "robots" 로 직접 기술

### Kling 강점 (느와르 시리즈에 유리)

- **고대비 조명 + 그림자 안정성**: 느와르 핵심인 로우키 조명, 하드 라이트 연출에 강함
- **복합 조명 해석**: `"soft key light from camera right mixed with warm practical lamp"` 같은 복합 지시를 정확히 처리
- **날씨·파티클**: 비, 안개, 조명 변화를 사실적으로 생성
- **카메라 언어**: `"anamorphic lens, 24mm, f/2.8"` 등 기술적 카메라 용어가 스타일 큐로 작동
- **카메라 움직임 (v3.0)**: 팬, 줌, 트래킹, 돌리 등이 안정적으로 생성

### Kling 제한 사항 (워크어라운드 필요)

| 제한 | 워크어라운드 |
|------|-------------|
| 눈/손/색상 플리커 발생 가능 | 고양이 캐릭터 눈 렌더링 주의. 레퍼런스 이미지 필수 |
| 빠른 카메라 움직임 시 아티팩트 | 샷당 카메라 움직임 1개로 제한 |
| 상충되는 조명 지시 혼란 | 샷당 조명 환경 1개로 통일 |
| 애니메/스타일라이즈드 캐릭터 약함 | **Image-to-Video 워크플로우 필수** (텍스트만으로 고양이 얼굴+인간 몸 일관성 불가) |
| 복잡한 물리 동작 왜곡 | 동작은 단순하게. 감정은 표정으로 |
| 클립 최대 5~15초 (v3.0) | 에피소드 = 5~8개 세그먼트로 분할 조립 |

### 캐릭터 일관성 4단계 아키텍처

| 단계 | 구성 요소 | 용도 |
|------|----------|------|
| 1 | **Character DNA** (문서) | 실루엣, 얼굴 앵커, 시그니처 소품, 스타일 락, DO NOT ALLOW 목록 |
| 2 | **Character Pack** (이미지) | 전신 4방향 + 표정 시트 6개 + 액션 포즈 3+ |
| 3 | **Shot Keyframe** (정지 이미지) | 각 샷의 구도를 정지 이미지로 먼저 확정 |
| 4 | **Animation** (Image-to-Video) | 키프레임에 동작만 부여. 캐릭터 외형은 이미지가 결정 |

핵심: **Image-to-Video 모드에서는 동작과 카메라만 프롬프트로 지시**. 캐릭터 외형은 입력 이미지가 결정.

### Elements 기능 (Kling v1.6+)

- 레퍼런스 이미지 1~4장 동시 업로드 → 시각적 정체성 추출·유지
- v3.0 Omni: 3~8초 캐릭터 영상에서 외형+음성 추출, 재사용 가능한 엘리먼트 생성
- 복수 엘리먼트 조합 시 교차 오염 없이 각각의 특징 유지

### 샷 타입별 용도와 안정성

| 샷 타입 | 용도 | Kling 안정성 |
|---------|------|-------------|
| Wide/Establishing | 장소·분위기 설정 | 높음 — 환경 생성 강점 |
| Medium | 캐릭터+맥락, 대화 | 높음 — 피사체 1명 시 안정 |
| Close-up | 감정, 반응, 디테일 | 중간 — 얼굴 아티팩트 주의 |
| Extreme Close-up | 소품, 극적 강조 | 높음 — 디테일 생성 우수 |
| Over-the-shoulder | 대화, 시점 | 중간 — 전환/이음새 숨기기에 유용 |

### 카메라 움직임 안전도

| 움직임 | 안전도 | 용도 |
|--------|--------|------|
| Static hold | 가장 안전 | 대화, 긴장, 강조 |
| Slow push-in | 안전 | 감정 드러남, 친밀감 |
| Slow pull-out | 안전 | 리빌, 고립, 엔딩 |
| Tracking (lateral) | 보통 | 피사체 따라감 |
| Pan | 보통 | 환경 훑기 |
| Low-angle | 안전 | 위압, 권력, 위협 |
| High-angle | 안전 | 취약, 감시 |
| Crane/Jib | 위험 | 수직 리빌 — 아티팩트 가능 |
| 빠른 회전 | 매우 위험 | 사용 자제 |

### 분위기별 키워드 뱅크

**비**: `rain-soaked`, `rain streaks visible against backlight`, `wet pavement reflections`, `rain pattering against fogged window`

**네온**: `neon signs casting [색상] reflections`, `neon bokeh`, `neon reflections in puddles`, `backlit by neon`

**따뜻한 조명**: `warm practical lamp light`, `amber glow`, `warm golden tones`, `tungsten warmth`

**안개/연기**: `light fog at ground level`, `smoke wisps curling`, `volumetric fog`, `haze diffusing light`

**그림자**: `deep shadows`, `chiaroscuro lighting`, `venetian blind shadows`, `face partially in shadow`, `silhouette against [광원]`

### 에피소드 샷 구성 (40~50초 기준)

7~8개 세그먼트 (각 5~10초):
1. Establishing wide (5s) — 장소·분위기
2. Medium 캐릭터 A (5s) — 주인공 소개
3. Close-up 반응/디테일 (5s) — 감정 비트
4. Medium two-shot 또는 OTS (5s) — 상호작용
5. Close-up 캐릭터 B (5s) — 상대 반응
6. Medium 액션 비트 (5~10s) — 핵심 드라마틱 순간
7. Wide 또는 medium 풀백 (5s) — 해결/클리프행어

### 스타일 락 (모든 프롬프트에 고정)

접미사:
```
cinematic, shallow depth of field, film noir aesthetic,
desaturated color palette, high contrast, low-key lighting,
rain-slicked surfaces, anamorphic lens, professional color grading
```

핵심 규칙: **스타일 어휘를 바꿔 쓰지 않는다.** "film noir aesthetic"을 "dark detective mood"로 의역하면 일관성이 깨진다.

### 제작 시간 추정

캐릭터 팩·환경 라이브러리 구축 후 기준:
- 에피소드 1편 (40~50초, 7~8샷): 약 90~120분
- 키프레임 생성: 30~45분
- 애니메이션: 20~30분
- 조립/폴리시: 10~15분
